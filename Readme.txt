Notre Projet est une version révisitée du jeu Motus. Ce jeu de mot consiste à deviner des mots en fonctions d'un nombre de lettres bien défini, choisi aléatoirement entre 6 et 10.

Il y'aura donc possibilité de jouer seul ou à deux en fonction des desirs tout un chacun.


***********************************Mise en route******************************************


Pour mettre en route notre application vous devez :

Démarrer la classe PartieMVC() avec les paramètres "gui" pour la vue interface graphique et "console" pour la vue en console
java PartieMVC.class console ou java PartieMVC.class gui

Dans le cas où le nombre de joueurs est égal à deux, vous devez démarrer aussi la classe Client() qui va servir d'interface pour le deuxième joueur. Ceci n'est possible uniquement pour la vue en console.
Suivre ensuite les instructions affichées par le terminal de la console pour jouer au jeu.
